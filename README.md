# 📊 Predicción de Churn en Telecomunicaciones (MLP + PCA)
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Enrique-Work/Churn-Prediction-MLP-PCA/blob/main/ModelosRN.ipynb)

Este proyecto implementa una **Red Neuronal Multicapa (MLP)** construida desde cero utilizando únicamente **NumPy** para predecir la probabilidad de fuga de clientes (*Churn*). El enfoque principal es entender la matemática detrás del aprendizaje profundo sin librerías externas de ML.

## 💡 Aspectos Técnicos Destacados
* **Arquitectura de Red:** Implementación manual de *Forward Propagation* y *Backpropagation*.
* **PCA (Análisis de Componentes Principales):** Reducción de dimensionalidad a 2 componentes para visualización de fronteras de decisión.
* **Preprocesamiento:** Limpieza de datos, codificación de variables categóricas y normalización.
* **Visualización:** Gráficas 2D de la separación de clases y matrices de confusión detalladas.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** Python
* **Librerías Core:** NumPy, Pandas
* **Visualización:** Matplotlib, Seaborn

## 📈 Resultados
El modelo alcanzó un **Accuracy del 77.77%**. La visualización permite identificar claramente los patrones de comportamiento de los clientes que deciden abandonar el servicio.
